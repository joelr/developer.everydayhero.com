Charities

List all Charities

GET /charities

View demo in API console

Visibility

Public.

Parameters

campaign_ids
Optional string. A comma-separated list of Campaign IDs, only Charities participating in the specified Campaigns will be retrieved.

Response

{
  "charities": [
    {
      "id": "au-1",
      "name": "Starlight",
      "slug": "starlight",
      "country_code": "au",
      "description": null,
      "gift_aid": false,
      "logo_url": "http://"
    }
  ]
}
